## 1. 実験目標

本実験の中核的な問いは次の一文に集約される。

> **座標降下法（CD_B）が精度 ε に到達するまでに要する更新回数は、次元 d に依存するか？**

より正確には、次の量を測定する。

$$t_\varepsilon := \min \{ t \mid f(B^{(t)}) - f(B^*) \le \varepsilon \}$$

ここで $f$ は DAG 構造学習の目的関数（対数尤度の負値）、$B^*$ は真の重み行列、
$t$ は**単一座標更新**を 1 単位とするカウンターである。

実験を通じて検証したいのは、次の「次元独立性」の有無である。

$$t_\varepsilon \approx O(1) \quad \text{(dimension-free)} \quad \text{vs} \quad t_\varepsilon \propto d \quad \text{or} \quad d^2$$

---

## 2. 理論背景

### 2.1 標準的な座標降下法の理論

滑らかな強凸関数 $f$ に対して、ランダム座標降下法の期待収束率は一般に

$$\mathbb{E}[f(x^{(t)}) - f(x^*)] \le \left(1 - \frac{\mu}{d L}\right)^t$$

で与えられる（$\mu$：強凸定数、$L$：Lipschitz 定数）。
この形では $d$ が大きいほど収束が遅くなる（**次元依存性あり**）。

### 2.2 CD_B の特殊性

CD_B アルゴリズムでは、各座標 $(i,j)$ に対して**閉形式の最適ステップ幅** $\delta^*_{ij}$ が利用できる。

$$\delta^*_{ij} = \arg\min_\delta f_B(B + \delta e_{ij})$$

これにより

- 各座標更新が**ちょうど 1 次元の厳密最小化**になる
- Lipschitz 定数が座標間でほぼ均一になる可能性がある

これらの性質が重なると、$O(d)$ の次元依存項が打ち消され、
**次元数によらないエポック数**での収束が期待できる。

## 3. 実験設計

### 3.1 設定の概要

| パラメータ | 値 |
|---|---|
| 次元 $d$ | 5, 10, 20, 40, 80 |
| 1 次元あたり試行回数 $n$ | 50 |
| 初期ノイズスケール $\sigma$ | 1.0 |
| 収束閾値 $\varepsilon$ | $10^{-3}$ |
| 最大更新回数 | $500 \times d^2$ |
| 座標選択方式 | ランダム一様（upper triangle） |
| 乱数シード | 42 |

### 3.2 真の解 $B^*$ の生成

各試行において、次の手順でランダムな真の DAG を生成する。

1. **トポロジー順序を固定**して上三角行列 $B^* \in \mathbb{R}^{d \times d}$ を構成
2. 各エントリ $(i,j)\ (i<j)$ に対し、辺確率 $p = 2/d$ で辺を追加
   （期待辺数 $\approx d$、スパースグラフ）
3. 辺重みは $U[0.5, 2.0]$ からサンプルし、符号を等確率で正負に設定

### 3.3 母集団共分散行列（サンプリングノイズの排除）

DAG 線形モデル $X = {B^*}^\top X + N$（$N \sim \mathcal{N}(0, I)$）において、
サンプル数 $n \to \infty$ の極限の共分散行列は

$$S = (I - {B^*}^\top)^{-1}(I - B^*)^{-\top}$$

この $S$ を用いることで、**サンプリングノイズを完全に排除**し、
アルゴリズム自体の収束性のみを評価できる。

この設定では $B^*$ が上三角なので $\det(I - {B^*}^\top) = 1$ が成り立ち、
最適目的値は

$$f(B^*) = d$$

と解析的に与えられる。

### 3.4 初期点の設計（初期距離の次元不変性）

初期距離が $d$ に依存しないよう、次のノイズスキームを採用する。

$$B^{(0)}_{ij} = B^*_{ij} + \epsilon_{ij}, \quad \epsilon_{ij} \sim \mathcal{N}\!\left(0,\, \frac{\sigma^2}{d^2}\right) \quad (i < j)$$

上三角エントリ数は $\frac{d(d-1)}{2} \approx \frac{d^2}{2}$ なので、

$$\mathbb{E}\!\left[\|B^{(0)} - B^*\|_F^2\right] \approx \frac{d^2}{2} \cdot \frac{\sigma^2}{d^2} = \frac{\sigma^2}{2}$$

すなわち **$\|B^{(0)} - B^*\|_F \approx \sigma/\sqrt{2}$ は次元によらず一定**となる。

### 3.5 アルゴリズムの実行

各試行において、以下のループを実行する。

```
t = 0
while gap > ε and t < max_updates:
    (i, j) = uniform_random(upper_triangle_coords)
    δ* = closed_form_optimal_step(B, S, i, j)   # delta_star_B
    B[i,j] += δ*
    M_inv を Sherman-Morrison で更新
    t += 1
    if t % d == 0: gap = f_B(B, S) - f_star
```

- `check_interval = d`（$d$ 回ごとに目的関数を評価、計算コストを削減）
- `M_inv` の更新は Sherman-Morrison 公式で $O(d^2)$、失敗時のみ完全逆行列を再計算

## 4. 評価指標

| 指標 | 定義 | 目的 |
|---|---|---|
| $t_\varepsilon$ | 収束に要した座標更新回数 | 生の次元依存性を直接観察 |
| **エポック数** $= t_\varepsilon / (d(d-1)/2)$ | $t_\varepsilon$ を座標数で正規化 | エポックレベルの次元独立性を評価 |
| **スケーリング指数** $\alpha$ | $\log t_\varepsilon \sim \alpha \log d$ の傾き | 次元依存性の強さを定量化 |
| 収束率 | $n_\text{converged} / n_\text{repeats}$ | 信頼性の確認 |

**基準となるスケーリング**

| 挙動 | $\alpha$（生） | エポック指数 | 解釈 |
|---|---|---|---|
| 完全次元独立 | $2.0$ | $0.0$ | 理論的な理想 |
| 線形スケール | $3.0$ | $1.0$ | 典型的な CD |
| 二乗スケール | $4.0$ | $2.0$ | スケーリング不良 |

## 5. 実験結果

### 5.1 収束統計（全試行収束率 = 100%）

| $d$ | mean $t_\varepsilon$ | std | median $t_\varepsilon$ | IQR | **mean epochs** |
|---|---|---|---|---|---|
| 5 | 157 | 200 | 110 | [50, 169] | **15.7** |
| 10 | 2,387 | 3,677 | 1,530 | [760, 2,318] | **53.0** |
| 20 | 10,536 | 7,269 | 8,320 | [5,595, 14,425] | **55.5** |
| 40 | 72,798 | 77,915 | 50,100 | [32,920, 65,050] | **93.3** |
| 80 | 305,922 | 203,258 | 245,640 | [183,220, 331,720] | **96.8** |

### 5.2 スケーリング分析

回帰 $\log t_\varepsilon \sim \alpha \log d$ から得られた結果：

| 対象 | フィット指数 $\alpha$ | 解釈 |
|---|---|---|
| 生の更新回数 $t_\varepsilon$ | **2.68** | $d^2$ よりやや大きい（$d^{2.68}$） |
| エポック数（$t_\varepsilon / n_\text{coords}$） | **0.61** | $d^1$ より大幅に小さい（準線形） |

### 5.3 局所スケーリング指数の推移

| $d$ の区間 | エポック指数（局所） | 傾向 |
|---|---|---|
| $d=5 \to 10$ | ≈ 1.75 | 初期は急上昇（$d=5$ が特異点） |
| $d=10 \to 20$ | ≈ 0.07 | ほぼ平坦 |
| $d=20 \to 40$ | ≈ 0.75 | 緩やかな上昇 |
| $d=40 \to 80$ | ≈ 0.06 | ほぼ平坦 |

## 7. 結論と解釈

### 7.1 主要な発見

1. **全次元で収束率 100%**
   $d=5$ から $d=80$ まで 50 試行すべてで $\varepsilon=10^{-3}$ に収束した。

2. **生の更新回数は $O(d^{2.68})$**
   単純に更新回数を比較すると $d$ に対してやや超二乗的に増加する。
   しかしこれは「各エポックに $d^2/2$ 回の更新が必要」という自明な部分を含む。

3. **エポック数のスケーリングは $d^{0.61}$（準線形）**
   更新回数を座標数で割ったエポック数に注目すると、$d^1$ よりはるかに緩やかに増加する。
   特に $d=10 \to 20$ および $d=40 \to 80$ の区間では局所指数が 0.07 と 0.06 であり、
   ほぼ平坦である。

4. **$d=5$ は特異点**
   エポック数が 15.7 と異常に小さい。小さな $d$ ではグラフがほぼ完全グラフになり、
   問題の性質が大きな $d$ と異なることが原因と考えられる。
   $d \ge 10$ の範囲に限れば指数はより安定している。

### 7.2 解釈のフレームワーク

| 実験で観測された量 | 理論的含意 |
|---|---|
| エポック指数 ≈ 0.61（全体） | $t_\varepsilon = O(d^{0.61}) \times O(d^2/2)$ = $O(d^{2.61})$ → 実測 $d^{2.68}$ と整合 |
| $d=10\to20$、$d=40\to80$ で局所指数 $\approx 0$ | **特定の次元帯では epoch-dimension-free が成立** |
| 閉形式の $\delta^*$ の利用 | 各座標更新が完全な 1D 最小化となり、有効な Lipschitz 定数が揃う可能性 |

### 7.3 残課題

- $d = 5$ の異常エポック数の原因分析（グラフ密度、問題の条件数）
- $d = 100, 200$ での外挿実験（局所指数がさらに 0 に収束するかの検証）
- $\Omega$（ノイズ共分散）が単位行列でない場合への一般化
- CD_BOmega との比較：ノイズ構造を陽に考慮した場合の次元独立性
